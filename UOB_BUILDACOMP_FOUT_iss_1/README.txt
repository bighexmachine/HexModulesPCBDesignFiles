Description of files included in UOB_BUILDACOMP_FOUT_iss_1_order.zip:

./README.txt    This File

./gerb/   This directory contains the following gerber files for the design.

4bit4wayFanOut-F_Cu.gtl	Top Copper Layer 	(Layer 1)
4bit4wayFanOut-B_Cu.gbl	Bottom Copper Layer 	(Layer 2)
4bit4wayFanOut-F_Mask.gts	Top Solder Mask
4bit4wayFanOut-B_Mask.gbs	Bottom Solder Mask
4bit4wayFanOut-F_Paste.gtp	Top Solder Paste
4bit4wayFanOut-B_Paste.gtp	Back Solder Paste
4bit4wayFanOut-F_SilkS.gto	Top Silk Screen
4bit4wayFanOut-B_SilkS.gto	Back Silk Screen
4bit4wayFanOut-drl_map.pho	Gerber format Drill File (plated through holes)
4bit4wayFanOut-NPTH-drl_map.pho   Gerber format Drill File (non-plated through holes)
4bit4wayFanOut-Edge_Cuts.gbr Edge Cuts

** It is acceptable for the holes marked "non-plated through hole" to be plated, if this aids manufacturing **


./drill

4bit4wayFanOut.drl	Excellon format drill file, plated holes - Decimal, absolute co-ordinates. Hole size is finished size.
4bit4wayFanOut-NPTH.drl   Excellon format drill file, non-plated holes (mechanical) - Decimal, absolute co-ordinates. Hole size is finished size.
4bit4wayFanOut-drl.rpt	Text drill size and number summary


./BOM

4bit4wayFanOut-BOM.csv	Excel format Bill of Materials


./XY/			XY Coponent position data in Excel format
4bit4wayFanOut-XY_Positions-F.Cu.csv    Front positions file
4bit4wayFanOut-XY_Positions-B.Cu.csv    Back positions file

./netlist/      This directory contains the netlist files for equivalence checking and bare board test.

4bit4wayFanOut.net	Netlist (KiCad format)


./schematics Schematic in .pdf format

4bit4wayFanOut.pdf

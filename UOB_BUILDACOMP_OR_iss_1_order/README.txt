Description of files included in UOB_BUILDACOMP_OR_order.zip:

./README.txt    This File

./gerb/   This directory contains the following gerber files for the design.

or_v2_passthrough_pulldowns-F_Cu.gtl	Top Copper Layer 	(Layer 1)
or_v2_passthrough_pulldowns-B_Cu.gbl	Bottom Copper Layer 	(Layer 2)
or_v2_passthrough_pulldowns-F_Mask.gts	Top Solder Mask
or_v2_passthrough_pulldowns-B_Mask.gbs	Bottom Solder Mask
or_v2_passthrough_pulldowns-F_Paste.gtp	Top Solder Paste
or_v2_passthrough_pulldowns-B_Paste.gtp	Back Solder Paste
or_v2_passthrough_pulldowns-F_SilkS.gto	Top Silk Screen
or_v2_passthrough_pulldowns-B_SilkS.gto	Back Silk Screen
or_v2_passthrough_pulldowns-drl_map.pho	Gerber format Drill File (plated through holes)
or_v2_passthrough_pulldowns-NPTH-drl_map.pho	Gerber format Drill File (non-plated through holes)
or_v2_passthrough_pulldowns-Edge_Cuts.gbr	Edge Cuts

** It is acceptable for the holes marked "non-plated through hole" to be plated, if this aids manufacturing **


./drill

or_v2_passthrough_pulldowns.drl	Excellon format drill file, plated holes - Decimal, absolute co-ordinates. Hole size is finished size.
or_v2_passthrough_pulldowns-NPTH.drl	Excellon format drill file, non-plated holes (mechanical) - Decimal, absolute co-ordinates. Hole size is finished size.
or_v2_passthrough_pulldowns-drl.rpt	Text drill size and number summary


./BOM

or_v2_passthrough_pulldowns-BOM.csv	CSV format Bill of Materials


./XY/			XY Component position data in Excel format
or_v2_passthrough_pulldowns-XY_Positions-F.Cu.csv	Front positions file

./netlist/      This directory contains the netlist files for equivalence checking and bare board test.

or_v2_passthrough_pulldowns.net	Netlist (KiCad format)


./schematics

or_v2_passthrough_pulldowns.pdf	Schematic in .pdf format

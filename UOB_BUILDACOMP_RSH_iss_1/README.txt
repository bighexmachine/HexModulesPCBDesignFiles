Description of files included in UOB_BUILDACOMP_RSH_iss_1_order.zip:

./README.txt    This File

./gerb/   This directory contains the following gerber files for the design.

4bit0123RightShifter-F_Cu.gtl	Top Copper Layer 	(Layer 1)
4bit0123RightShifter-B_Cu.gbl	Bottom Copper Layer 	(Layer 2)
4bit0123RightShifter-F_Mask.gts	Top Solder Mask
4bit0123RightShifter-B_Mask.gbs	Bottom Solder Mask
4bit0123RightShifter-F_Paste.gtp	Top Solder Paste
4bit0123RightShifter-B_Paste.gtp	Back Solder Paste
4bit0123RightShifter-F_SilkS.gto	Top Silk Screen
4bit0123RightShifter-B_SilkS.gto	Back Silk Screen
4bit0123RightShifter-drl_map.pho	Gerber format Drill File (plated through holes)
4bit0123RightShifter-NPTH-drl_map.pho   Gerber format Drill File (non-plated through holes)
4bit0123RightShifter-Edge_Cuts.gbr Edge Cuts

** It is acceptable for the holes marked "non-plated through hole" to be plated, if this aids manufacturing **


./drill

4bit0123RightShifter.drl	Excellon format drill file, plated holes - Decimal, absolute co-ordinates. Hole size is finished size.
4bit0123RightShifter-NPTH.drl   Excellon format drill file, non-plated holes (mechanical) - Decimal, absolute co-ordinates. Hole size is finished size.
4bit0123RightShifter-drl.rpt	Text drill size and number summary


./BOM

4bit0123RightShifter-BOM.csv	Excel format Bill of Materials


./XY/			XY Component position data in Excel format
4bit0123RightShifter-XY_Positions-F.Cu.csv    Front positions file (Assembly on front side only)

./netlist/      This directory contains the netlist files for equivalence checking and bare board test.

4bit0123RightShifter.net	Netlist (KiCad format)


./schematics Schematic in .pdf format

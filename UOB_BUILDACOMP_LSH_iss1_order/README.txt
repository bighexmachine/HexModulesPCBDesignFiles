Description of files included in UOB_BUILDACOMP_LSH_iss1_order.zip:

./README.txt	This File

./gerb/	This directory contains the following gerber files for the design.

4bit0123LeftShifter-F_Cu.gtl	Top Copper Layer	(Layer 1)
4bit0123LeftShifter-B_Cu.gbl	Bottom Copper Layer	(Layer 2)
4bit0123LeftShifter-F_Mask.gts	Top Solder Mask
4bit0123LeftShifter-B_Mask.gbs	Bottom Solder Mask
4bit0123LeftShifter-F_Paste.gtp	Top Solder Paste
4bit0123LeftShifter-B_Paste.gbp	Bottom Solder Paste
4bit0123LeftShifter-F_SilkS.gto	Top Silk Screen
4bit0123LeftShifter-B_SilkS.gbo Bottom Silk Screen
4bit0123LeftShifter-drl_map.pho	Gerber format Drill File	(plated through holes)
4bit0123LeftShifter-NPTH-drl_map.pho	Gerber format Drill File	(non-plated through holes)
4bit0123LeftShifter-Edge_Cuts.gbr	Gerber format edge cuts	(PCB outline)

** It is acceptable for the holes marked "non-plated through hole" to be plated, if this aids manufacturing **


./drill

4bit0123LeftShifter.drl	Excellon format drill file, plated holes - Decimal, absolute co-ordinates. Hole size is finished size.
4bit0123LeftShifter-NPTH.drl	Excellon format drill file, non-plated holes (mechanical) - Decimal, absolute co-ordinates. Hole size is finished size.
4bit0123LeftShifter-drl.rpt	Text drill size and number summary


./BOM

4bit0123LeftShifter-BOM.csv	CSV format Bill of Materials


./XY

4bit0123LeftShifter-F.Positions.csv	CSV format positions file (components present on front/top layer only)


./netlist

4bit0123LeftShifter.net	Netlist for equivalence checking and bare board test (KiCad format)


./schematics

4bit0123LeftShifter-Schematic.pdf PCB schematic in .pdf format for reference
